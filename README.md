# BackpackDemo

> Unity 客户端验证项目。hermes-harness 工作线的 Unity 验证场。

## 用途

在真实 Unity 项目中验证 Agent 管线能力 — 从需求拆分到代码实现到测试报告。

## 工作线报告

每次 Agent 任务完成后输出：

| 报告 | 说明 |
|:---|:---|
| SDD-*.md | 规格设计文档 |
| REVIEW.md | 独立审查报告 |
| TestReport.md | 测试结果 |
| ChangedFiles.md | 变更文件清单 |
| RiskReport.md | 风险评估 |
| ObsidianNote.md | Obsidian 知识沉淀 |

## 技术栈

- Unity 6000.3+
- C# / .NET Standard 2.1
